# Seorab
원하는 것을 자료형에 구애받지 않고 넣을 수 있는 창고형 웹사이트 "서랍"

## 프레임워크
- FE: React
- BE: Django

## 기능 목록

### 유저 인증
- 회원가입
- 로그인
  - 홈페이지 로그인
  - 소셜 연동 로그인
 - 로그아웃
 - 회원정보 수정
 - 회원 탈퇴
  
### 메인 화면
- CONTENTS 추가
  - TEXT
  - IMAGE
  - URL
- CONTENTS 드래그/드랍
- CONTECTS 정렬
  - 최신순
  - 가나다순
- CONTENTS 삭제
- 카테고리 추가
